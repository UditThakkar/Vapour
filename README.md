Vapour
=====

A game-store platform built with Ruby on Rails and Backbone.js.

###Features
-	AWS S3 file storage
-	On-the-fly filtering with tags
-	Dynamic modals
-	User authentication
-	Friending other users

###Features to Come
-	Messaging other users
-	Game Reviews (or Recommendations)
